+ 新しい用語を作成
  • AI（人工知能）
    たくさんのデータから自分でパターンを学び、決まったルールがない場面でも判断できるコンピューターの技術
  • API
    違うソフトウェア同士が、決まったルールでやり取りするための窓口
  • Bluetooth（ブルートゥース）
    ケーブルを使わずに、近くの機器同士を無線でつなぐ技術
  • Cookie（クッキー）
    サイトがあなたのブラウザに残す、小さなメモ（次に来たとき、あなたを覚えておくためのもの）
  • HTTP / HTTPS
    ブラウザとサイトが、ページをやり取りするときの共通のルール（手紙の書き方のようなもの）
  • Nintendo Switch
    テレビにつないでも、持ち歩いても遊べる任天堂のゲーム機（2017年発売）
  • OS（基本ソフト）
    ハードウェアとアプリの橋渡しをする、いちばん基本のソフトウェア
  • Sunshine Law（サンシャイン法）
    フロリダ州の法律で、行政の会議や記録を誰でも見られるようにする決まり
  • Wi-Fiとインターネットの違い
    Wi-Fiは近くの機器とつながる無線、インターネットはその先の世界中とつながる回線
  • いいとこ取り
    2つ以上のものから、それぞれの良いところだけを選んで組み合わせること
  • どんでん返し
    物語の結末や状況が、それまでの予想を大きく裏切って逆転すること
  • アプリ
    スマホやパソコンで使う、特定の目的のために作られたソフトウェアのこと
  • ウォルト・ディズニー・ワールド
    アメリカ・フロリダ州にある、世界最大級のテーマパークリゾート
  • エージェント
    自分で考えて、道具(ツール)を使いながら何段階も作業を進めてくれるAIのこと
  • クラウド
    データやアプリを、自分の端末ではなく、遠くにあるたくさんのコンピューターに置いておく仕組み
  • コンテキストウィンドウ
    AIが一度に覚えていられる会話量の上限のこと
  • コンピューター
    プログラムの指示通りに、計算や作業を行う機械のこと
  • サイト（ウェブサイト）
    インターネット上にある、会社やお店・個人の「住所」のようなページのまとまり
  • スーパーマリオ
    配管工のマリオが、ジャンプしながらいろんな世界を冒険する任天堂のゲームシリーズ（生みの親: 宮本茂）
  • ゼルダの伝説
    主人公リンクが、広い世界を探検しながら謎を解いていく任天堂のゲームシリーズ（生みの親: 宮本茂）
  • ソフトウェア
    コンピューターに「何をするか」を指示する、目に見えないプログラムのまとまり
  • データ
    コンピューターが扱う、文字や数字、写真、音声などの情報のこと
  • トークン
    AIが文章を読み書きするときに数える、言葉の最小単位
  • ハルシネーション
    AIが、間違った情報を正しい情報と同じくらい自信満々に話してしまうこと
  • ハードウェア
    画面や基盤など、実際に手で触れる機械の部分のこと（⇔ソフトウェア）
  • フロリダ
    アメリカ南東部にある、温暖な気候で知られる半島の州
  • ブラウザ
    インターネット上のサイトを見るための、専用のソフト
  • プログラム
    コンピューターに「何を、どの順番でするか」を指示する、命令の集まりのこと
  • ホモサピエンス
    生物学の分類で、今生きている人間（私たち）につけられた種としての正式な名前
  • ポケモン
    不思議な生き物「ポケットモンスター」を集めて育てる任天堂のゲームシリーズ（生みの親: 田尻智）
  • 京都
    神社やお寺が多く残る、日本の伝統文化の中心地となってきた都市
  • 任天堂
    マリオやゼルダ、Switchなどで知られる、日本のゲーム会社

  • 利息 / 年利

  • 匙を投げる

  • 台無し

    品質や実力が確かだと、はっきり保証できること
